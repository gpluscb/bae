instant-glicko-2

instant-glicko-2 is a Rust crate to calculate Glicko-2 ratings that update instantly after a game completes. Usually, you'd have to wait to the end of a rating period to have ratings update. I wrote about how to side-step that here, but that's mildly out of date now. The source code is on GitHub.

cache-compute

Another thing I stole from fasterthanlime? How wild is that. cache-compute is a Rust crate to cache results of async computations. I needed to cache some stuff for a personal project, but it turned out to be a harder problem than initially thought. If the cache is empty, you may want to start a computation. But you don't want to start one if someone else already did, you just want to be notified when that one is finished. Well luckily, at about the same time I // TODOd that, Amos published a very good article on the topic. And the implementation I created from that for my project grew decent enough to be factored out into a single-file crate, which I improved further and then open sourced on GitHub.
